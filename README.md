# Xelora Trading

Algorithmic trading strategies and market analysis framework. Built in Python with focus on backtesting accuracy and signal reliability.

## Features

- Strategy backtesting engine with realistic fee modeling
- Market regime detection and adaptive signal generation
- Portfolio-level risk management and position sizing
- Multi-exchange data aggregation and normalization

## Stack

- Python 3.11+
- pandas / numpy for data analysis
- FastAPI for signal serving
- WebSocket streaming for live market data
